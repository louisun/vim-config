## Install

```bash
mkdir ~/.config
git clone git://github.com/louisun/vim-config.git ~/.config/nvim
cd ~/.config/nvim

# for "regular" Vim
ln -s ~/.config/nvim ~/.vim

# install plugins
make install
```
